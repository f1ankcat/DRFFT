# DRFFT: A High-Performance Heterogeneous Fast Fourier Transform Framework

Welcome to the official repository for **DRFFT**. 

## 🚀 Coming Soon

We are currently organizing the source code, build configurations, and evaluation benchmarks for public release. The complete codebase will be made available here upon the formal publication of our accompanying paper.

### What to Expect
* **Optimi/zed Kernels:** High-performance FFT implementations supporting both power-of-2 and non-power-of-2 transform sizes.
* **Advanced Layout Strategies:** Custom memory mapping and dispatch mechanisms optimized for high-bandwidth heterogeneous architectures.
* **Reproduction Suites:** Complete end-to-end benchmarking scripts used to generate the latency and throughput comparisons reported in our paper.
